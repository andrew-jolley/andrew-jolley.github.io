Key Features

Excel Integration

Seamlessly export your lecture data to Excel spreadsheets with automatic formatting and organization.

KSB Mapping

Automatically maps module codes to Knowledge, Skills & Behaviours for apprenticeship tracking.

Dark/Light Theme

Choose between light and dark themes for comfortable use in any environment.

Smart Date Handling

Intelligent academic year calculation and date formatting for consistent data entry.

Customizable Settings

Configure Excel file paths, starting rows, logging preferences, and update settings.

Alpha Release Limitations

Early Development Stage

This alpha version includes core functionality but may be missing some features:

  • Limited error handling in some scenarios
  • Basic user interface (UI improvements planned)
  • Manual Excel file management required
  • No data backup or sync features yet
  • Windows version may require additional configuration

Ideal for: Students, researchers, and early adopters who want to test academic tracking tools.

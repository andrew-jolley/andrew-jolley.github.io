Key Features

Excel Integration

Seamlessly export your lecture data to Excel spreadsheets with automatic formatting and organization.

KSB Mapping

Automatically maps module codes to Knowledge, Skills & Behaviours for apprenticeship tracking.

Dark/Light Theme

Choose between light and dark themes for comfortable use in any environment.

Smart Date Handling

Intelligent academic year calculation and date formatting for consistent data entry.

Customizable Settings

Configure Excel file paths, starting rows, logging preferences, and update settings.

Built-in update checking system to keep you current with the latest features and fixes.
